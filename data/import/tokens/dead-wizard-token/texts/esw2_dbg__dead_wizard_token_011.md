sourceLabel: Каждый враг может показать тебе средний палец. Сбрось 1 карту за каждого кто это сделал

textRu:
Каждый враг может показать тебе средний палец. Сбрось 1 карту за каждого кто это сделал.

clarifications:

- В симуляции никто не будет показывать средний палец, просто потверждать или нет сброс карты у игрока
- В качестве шутки можно назвать функцию такую по сбросам карт за каждого согласного игрока "средний палец" (на англ языке)
- Каждый враг отдельно выбирает, делать ли это действие.
- За каждого врага, который это сделал, игрок должен сбросить 1 карту.
- Если несколько врагов сделали это, количество сбрасываемых карт равно количеству таких врагов.
- Если нужно сбросить 4 карты, а на руке меньше 4 карт, то на руке остаются 0 карт, так как случай с -1 или другим отрицательным числом карт невозможен
