sourceImages:

- assets/dead-wizard-token/DWT_001.png

sourceLabel: Получи вялую палочку за каждую легенду в твоей стопке сброса

textRu:
Получи вялую палочку за каждую легенду в твоей стопке сброса.

clarifications:

- Игрок получает вялые палочки.
- Количество вялых палочек равно количеству легенд в его стопке сброса.
- "Твоя стопка сброса" относится к игроку, получившему этот ЖДК.
- Есть маленькая вероятность ситуации, когда у игрока есть свойство [wp_003](../wizard-property-texts/wp_003.md), а в сбросе есть >0 фамильяров, то этот игрок может посчитать фамильяров за легенд, и получить вялые палочки за них. Польза от этого может быть если у игрока есть Легенда-Волшебник Виагрус, который меняет по вялой палочки с -1 на +1
