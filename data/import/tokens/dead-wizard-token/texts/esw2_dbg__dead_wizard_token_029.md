textRu:
Ты наносишь дополнительно 4 урона палочками, которые наносят урон.
-4 ПО

clarifications:

- Эффект увеличивает урон от палочек игрока.
- Увеличение применяется только к палочкам, которые действительно наносят урон.
- Значение -4 относится к штрафным ПО/VP этого жетона, если JSON mapping подтвердит это как scoring value.
- Фактически, этот ЖДК дает -7 ПО в сумме
- Дополнительный урон наносится независимо от владельца палочки, даже через шальную магию можно сыграть чужую палочку и нанести доп урон
