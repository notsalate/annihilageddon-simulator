# esw2_dbg\_\_wizard_property_004

sourceImage: assets/wizard-property/wp_004.png
sourceLabel: wp_004
textRu:
Ты платишь на 1 меньше при покупке сокровищ. В конце игры: каждое твоё сокровище приносит тебе +1 ПО.

clarifications:

- Скидка применяется при покупке сокровищ.
- При покупке легенды-сокровищ также применяется скидка.
- Стоимость покупки сокровища уменьшается на 1.
- В конце игры каждое сокровище игрока приносит дополнительно +1 ПО.
- ПО означает победные очки.
- Можно со старта свойства считать, что сокровища под владениям игрока уже дают +1 ПО
