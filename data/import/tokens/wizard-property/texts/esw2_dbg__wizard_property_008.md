# esw2_dbg\_\_wizard_property_008

sourceImage: assets/wizard-property/wp_008.png
sourceLabel: wp_008
textRu:
Каждый раз, когда ты играешь карту-постоянку, получай 1 чипсину. Каждый раз, когда ты получаешь карту-постоянку, можешь класть её на верх своей колоды.

clarifications:

- Первая часть срабатывает каждый раз, когда игрок играет карту-постоянку.
- Игрок сыграл карту-постоянку, когда в первый раз сыграл эту карту, точнее положил ее в поле для постоянок. Последущие применения свойств активируемой постоянки не подходят под условие свойства. Только если постоянка была каким-то образом сброшена, и позже, заново активирована
- Легенды со свойством "Постоянка" подходят под условие свойства
- За каждую сыгранную карту-постоянку игрок получает 1 чипсину.
- Вторая часть срабатывает каждый раз, когда игрок получает карту-постоянку.
- Полученную карту-постоянку игрок может положить на верх своей колоды.
- Слово "можешь" означает, что размещение карты на верх колоды необязательно.
- Постоянка купленная/полученная в "сброс" не может быть поставлена на верх колоды по свойству.
