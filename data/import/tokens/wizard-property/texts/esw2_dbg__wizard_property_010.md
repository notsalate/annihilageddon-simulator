# esw2_dbg\_\_wizard_property_010

sourceImage: assets/wizard-property/wp_010.png
sourceLabel: wp_010
textRu:
Начни игру с главным призом. Ты ходишь первым. Если ты не лошара, начальное значение твоих жизней (даже при воскрешении) равно 25.

clarifications:

- Это свойство применяется при подготовке к игре.
- Игрок начинает игру с главным призом.
- Игрок должен ходить первым.
- Начальные значения жизней игрока равны 25.
- Уточнение "даже при воскрешении" означает, что значение 25 также учитывается при воскрешении после смерти.
- Если игрок получил жетон Лошары, то его максимум жизней - 15
