## What to build

Реализовать первый Block D runtime implementation slice: весь кластер `simple-baseline`.

Срез должен довести все 7 выбранных карт до `fullRuntime`: добавить текущие runtime card definitions, включить карты в соответствующие current compositions, покрыть поведение focused tests и обновить generated matrix.

Это не foundation issue. Не выносить заранее общие attack, defense или другие широкие runtime primitives. Если для этих простых карт нужен общий helper, вводить его только в объёме этого slice.
