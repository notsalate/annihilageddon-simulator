# Ввести fullRuntime audit guardrails и убрать stale coverage snapshot

## Parent

- `.scratch/krutagidon-card-runtime-clusters/PRD.md`

## What to build

Довести card cluster report до full-only guardrail: отчёт должен говорить языком `fullRuntime`, находить invalid references, не допускать новые partial runtime cards и заменить старый stale docs snapshot.

Срез должен замкнуть Block B: после него агент может запускать card-cluster report перед и после card implementation slices, а committed generated docs snapshot больше не является workflow artifact.

## User stories covered

- 5. Blocked cards kept out of runtime JSON.
- 6. No new partial runtime cards.
- 10. Autogenerated matrix does not rot.
- 11. Commands to refresh reports.
- 13. Invalid runtime/composition references detected.

## Acceptance criteria

- [ ] Card cluster report использует статус `fullRuntime` для карт, у которых есть runtime JSON, composition membership и проверяемые evidence/test refs.
- [ ] Старые двусмысленные статусы не являются целевым языком нового card workflow.
- [ ] Report обнаруживает decisions, которые ссылаются на несуществующие card drafts.
- [ ] Report обнаруживает card drafts без decisions.
- [ ] Report обнаруживает composition entries, которые ссылаются на отсутствующие runtime card definitions.
- [ ] Report обнаруживает runtime card JSON без matching draft.
- [ ] Report явно падает или выдаёт blocking error для новых card runtime JSON с partial/non-full признаками.
- [ ] Report не считает `missingRuntime` ошибкой процесса.
- [ ] `docs/runtime-coverage-v05-inventory.md` удалён как stale generated snapshot.
- [ ] Документация или local AGENTS объясняет, что dynamic command и generated matrix заменяют старый committed coverage markdown.
- [ ] Focused tests покрывают `fullRuntime`, invalid decision, invalid composition reference и partial runtime guardrail.
- [ ] `npm run report:card-runtime-clusters` проходит на текущем clean baseline.
- [ ] `npm run typecheck` проходит.
- [ ] Focused generator tests проходят.
- [ ] `git diff --check` проходит.
